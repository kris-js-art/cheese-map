# Cheese Map 🧀

Сервис для управления информацией о сырах, производителях, категориях, магазинах и отзывах. Проект построен на Spring Boot и предоставляет полный набор CRUD операций для управления каталогом сыров.

## Технологии

- **Java 21**
- **Spring Boot 4.x**
- **Spring Web MVC**
- **Spring Data JPA**
- **Lombok**
- **MapStruct**
- **PostgreSQL**
- **Maven**

## Функциональность

### Основные сущности:
- **Cheese (Сыр)** - основная сущность с характеристиками
- **Producer (Производитель)** - компании, производящие сыр
- **Category (Категория)** - типы и классификация сыров
- **Shop (Магазин)** - точки продажи сыров
- **Review (Отзыв)** - отзывы покупателей

### Возможности:
- Полное CRUD управление для всех сущностей
- Связи между сущностями (многие-ко-многим, один-ко-многим)
- Поиск по различным критериям
- Транзакционное создание связанных объектов
